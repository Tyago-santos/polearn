# Polearn

![Logo do Polearn](public/logo.png)

Polearn e um app web para aprender diversos idiomas de forma pratica e intuitiva.

## Objetivo do app

O objetivo do Polearn e ajudar pessoas a aprender idiomas com consistencia, por meio de uma experiencia simples e objetiva. A proposta e transformar o estudo em uma rotina leve, com atividades praticas para leitura e escrita, , permitindo evolução continua no dia a dia.

## Tecnologias

- React 19
- TypeScript
- Vite
- TanStack Router
- Tailwind CSS 4
- Lucide React
- ESLint

## Como iniciar o projeto

1. Instale as dependencias:

```bash
npm install
```

2. Rode em modo de desenvolvimento:

```bash
npm run dev
```

3. Gere a build de producao:

```bash
npm run build
```

4. Visualize a build localmente:

```bash
npm run preview
```

5. Rode a analise de codigo:

```bash
npm run lint
```
